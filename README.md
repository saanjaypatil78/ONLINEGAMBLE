# Matka Solana MVP

Matka Solana is a skill-based lottery prototype that pairs players, settles outcomes on Solana, and distributes rewards automatically. This landing page keeps you oriented while the full story lives in our guided documentation.

## Start Here

- Read the non-technical orientation: [`docs/matka-solana-guide.md`](docs/matka-solana-guide.md:1)

## Quick Start Commands

```bash
npm install
./scripts/bootstrap.sh
```

Run these from the repository root to install workspace dependencies and follow the bootstrap prompts. Continue with the Matka Solana Guide for environment setup, deployment checklists, and day-to-day operations.
